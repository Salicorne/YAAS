YAAS Project

This website is a graded project for the course "DT00BT67 : Development of server-side web services" at Åbo Akademi University. It is an example of the YAAS application : Yet Another Auction Website.

The sources of this project can be found here. The principle of the YAAS application is to create and participate in auctions. This application relies on the Python web framework Django, and handles user accounts creation, authentification, authorization and edition, auction creation and bidding.

This application is available at http://traboeuf.pythonanywhere.com or https://yaas.salicorne.ovh.
